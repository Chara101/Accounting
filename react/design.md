# Design
This file is for ai to generate content.
### 核心功能
* CRUD
* 每筆記錄在總覽全部與篩選後頁面都只顯示科目、子科、金額、備註，為概覽
* 每筆紀錄的概覽都能點進去，跳出一個頁面用來修改紀錄
* 有頁面用來篩選和顯示篩選結果
* 篩選:按照月份或科目或子科目查看總金額、紀錄筆數、紀錄
* 新增與刪除科目與子科目
mvp: 只做全部瀏覽、加入與刪除資料

狀態管理：React內建
畫面：框架隨意，一頁最多展示10條紀錄概覽
路由：
1. 總覽畫面，包含總金額、全部紀錄概覽，概覽以天為一個小群組框起來並顯示天總金額
2. 紀錄的詳細內容，包含使用下拉式修改科目及子科、user、金額、備註欄，可用於修改或新增紀錄時使用
3. 篩選
4. 新增紀錄
5. 科目、子科管理
